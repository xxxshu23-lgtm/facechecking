# 결과 공개 - 관상 팩트 체크

## 프로젝트 개요
한국 전통 관상학을 현대적인 UI/UX로 시각화한 웹페이지입니다. 
얼굴의 주요 부위(눈/눈썹, 코, 입/하관)를 투명 오버레이 방식으로 겹쳐서 표현했습니다.

## 현재 구현된 기능

✅ **메인 레이아웃**
- 크림색 배경(#f5f3ed)의 따뜻한 디자인
- "결과 공개" 세리프 타이틀 중앙 배치
- Noto Serif KR / Noto Sans KR 폰트 사용

✅ **이미지 오버레이 시스템**
- 메인 얼굴 일러스트레이션 (베이스 레이어)
- 눈/눈썹 상세 이미지 (투명 오버레이)
- 입/하관 상세 이미지 (투명 오버레이)  
- 코 상세 이미지 (투명 오버레이)
- CSS position: absolute로 완벽하게 겹침 처리
- 순차적 페이드인 애니메이션

✅ **네비게이션 버튼**
- **이전 버튼**: 이전 페이지로 이동
- **더 알아보기 버튼**: 상세 정보 페이지 (강조 스타일)
- **다음 버튼**: 다음 단계로 이동
- 호버 효과 및 트랜지션 애니메이션

✅ **반응형 디자인**
- 데스크톱: 최대 900px 너비 얼굴 이미지
- 태블릿: 버튼 레이아웃 유지
- 모바일: 세로 스택 버튼 배치
- 브레이크포인트: 768px, 480px

✅ **인터랙티브 요소**
- 버튼 호버 효과 (색상 변경, 그림자, 이동)
- 오버레이 이미지 순차 페이드인
- 부드러운 트랜지션 애니메이션

## 기술 스택
- **HTML5**: 시맨틱 마크업 (main, button)
- **CSS3**: Position absolute 오버레이, Flexbox, 반응형
- **JavaScript**: 이벤트 리스너, 애니메이션 컨트롤
- **Google Fonts**: Noto Serif KR, Noto Sans KR

## 파일 구조
```
/
├── index.html          # 메인 HTML 페이지
├── css/

├── js/
│   └── main.js         # 인터랙티브 기능

```

## 기능 URI
- **메인 페이지**: `/index.html` (또는 `/`)
  - 관상 분석 결과 화면
  - 투명 오버레이 이미지 표시
  - 네비게이션 버튼

## 디자인 특징

### 색상 팔레트
- **배경**: #f5f3ed (따뜻한 베이지)
- **주 색상**: #6b7c3d (올리브 그린)
- **호버 색상**: #8a9e5f (연한 그린)
- **텍스트**: #333 (다크 그레이), #999 (안내 문구)

### 타이포그래피
- **타이틀**: Noto Serif KR 600 (40px, letter-spacing: 10px)
- **버튼**: Noto Sans KR 500 (16px)
- **안내**: Noto Sans KR 400 (14px)

### 레이아웃 철학
- 투명 오버레이로 모든 상세 이미지를 메인 이미지 위에 겹침
- 원본 이미지의 투명도를 활용한 자연스러운 합성
- 깔끔한 중앙 정렬 레이아웃

## JavaScript 기능
- **더 알아보기 버튼**: Alert → 상세 페이지로 연결 예정
- **이전 버튼**: window.history.back()
- **다음 버튼**: 다음 단계 이동 (구현 예정)
- **오버레이 애니메이션**: 순차적 페이드인 (300ms 간격)

## 사용 방법
1. 웹 브라우저에서 `index.html` 파일 열기
2. 또는 **Publish 탭**에서 배포 후 라이브 URL 접속
3. 버튼을 클릭해 네비게이션 테스트

## UX 디자인 포인트
- **정보 계층**: 타이틀 → 이미지 → 버튼 → 안내문구
- **시각적 흐름**: 상단에서 하단으로 자연스러운 흐름
- **인터랙션**: 명확한 호버 피드백으로 클릭 유도
- **접근성**: 시맨틱 HTML, 충분한 터치 영역 (48px 이상)
