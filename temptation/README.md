# 관상 팩트 체크 웹사이트

## 프로젝트 개요
관상 분석 결과를 보여주고, 실제 사례 기사로 연결되는 인터랙티브 웹사이트입니다. 사용자는 "결과 공개" 페이지에서 얼굴 분석 결과를 확인하고, "더 알아보기" 버튼을 통해 관련 뉴스 기사로 이동할 수 있습니다.

## 주요 기능

### 현재 구현된 기능
1. **결과 공개 페이지 (index.html)**
   - 얼굴 일러스트와 확대 영역 표시
   - 눈 부분 확대 이미지 오버레이
   - "더 알아보기" 버튼으로 기사 페이지 이동
   - 부드러운 페이드 인/아웃 전환 효과

2. **뉴스 기사 페이지 (article.html)**
   - 기상캐스터 A씨 직장 내 괴롭힘 사건 기사
   - 가해자 이미지와 텍스트 레이아웃
   - 매칭 비교 섹션 (스케치 vs 실제 사진)
   - "관상 팩트 체크 완료" 스탬프 효과
   - 돌아가기 버튼으로 결과 공개 페이지 복귀

3. **인터랙션 기능**
   - 페이지 전환 시 부드러운 페이드 효과
   - 버튼 호버 시 애니메이션
   - 이미지 클릭 시 확대 효과
   - 스크롤 애니메이션 (기사 페이지)
   - 스탬프 등장 애니메이션

4. **반응형 디자인**
   - 데스크톱, 태블릿, 모바일 대응
   - 1920px 최대 너비 설정
   - 모든 디바이스에서 최적화된 레이아웃

## 페이지 구조 및 URI

### 메인 페이지
- **경로**: `index.html`
- **기능**: 관상 분석 결과 공개
- **주요 요소**: 얼굴 일러스트, 확대 박스, 네비게이션 버튼

### 기사 페이지
- **경로**: `article.html`
- **기능**: 실제 사례 뉴스 기사
- **주요 요소**: 기사 본문, 가해자 사진, 매칭 비교, 스탬프

### 네비게이션 흐름
```
index.html (결과 공개)
    ↓ [더 알아보기 버튼]
article.html (뉴스 기사)
    ↓ [돌아가기 버튼]
index.html (결과 공개)
```
