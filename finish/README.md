# 마무리 페이지

관상 분석 서비스의 마무리 페이지입니다. 전통적이면서도 모던한 디자인으로 사용자에게 따뜻한 메시지를 전달합니다.

## 🎨 완성된 기능

### 현재 구현된 기능
- ✅ 1920px 너비 기준 레이아웃 (반응형 지원)
- ✅ #F9F9F4 배경색 적용
- ✅ 전통적인 장식 라인 디자인 (상단/하단)
- ✅ 우아한 타이포그래피 (Noto Serif KR 폰트)
- ✅ "처음으로" 버튼 인터랙션
- ✅ 페이지 로드 애니메이션
- ✅ 버튼 호버 효과 및 전환 애니메이션
- ✅ 모바일 반응형 디자인

### 디자인 특징
- **배경색**: #F9F9F4 (베이지 톤)
- **포인트 컬러**: #8B956D, #A4AE7C (올리브 그린 계열)
- **레이아웃**: 900px 최대 너비의 중앙 정렬
- **폰트**: Noto Serif KR (세리프 계열로 전통적 느낌)
- **장식 요소**: 상하단 코너 장식 라인

## 📂 프로젝트 구조

```
├── index.html          # 메인 마무리 페이지
├── css/
│   └── style.css      # 스타일시트
├── js/
│   └── main.js        # JavaScript 인터랙션
└── README.md          # 프로젝트 문서
```

## 🎯 주요 기능 URI

- **메인 페이지**: `index.html`

## 🚀 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. "처음으로" 버튼 클릭 시 메인 페이지로 이동
3. 부드러운 페이드 효과와 함께 페이지 전환

## 💡 커스터마이징 가이드

### 버튼 동작 변경
`js/main.js` 파일의 `goToStart()` 함수에서 이동할 URL을 수정:
```javascript
window.location.href = '원하는페이지.html';
```

### 색상 변경
`css/style.css` 파일에서 다음 색상 코드 수정:
- **배경색**: `background-color: #F9F9F4`
- **포인트 컬러**: `#8B956D`, `#A4AE7C`
- **텍스트 컬러**: `#4A4A3A`

### 텍스트 수정
`index.html` 파일의 `.text-paragraph` 클래스 내용 수정
